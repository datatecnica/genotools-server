# Creating IAP credentials for each app
1. Go to Oauth Credentials screen in gcp console
2. Select: Create OAuth client ID
3. Select Web application
4. Add Authorized Redirect URI: https://URL/_gcp_iap/authenticate: https://genotracker-app.genotoolsservers.com/_gcp_iap/authenticate
